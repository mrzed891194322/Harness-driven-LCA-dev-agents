

## opencode commands

### 新建 RAG 知识库

opencode run --command init-rag-database

/init-rag-database

### 制定计划

opencode run --command make-plan "根据文件input\plan.md，制定新的LCA计划"

/make-plan “根据文件input\plan.md，制定新的LCA计划”

### 建立LCI

opencode run --command design-lci "根据计划src\plan\execution_plan.md，制定新的LCI"

/design-lci “根据计划src\plan\execution_plan.md，制定新的LCI”

### 导入 LCI

opencode run --command import-lci

/import-lci

## 脚本指令

### 清除文件夹中内容

uv run python docs/assets/scripts/clean_dir.py <要清理的目录相对路径> 

例如：

uv run python docs/assets/scripts/clean_dir.py src\plan 

uv run python docs/assets/scripts/clean_dir.py src

通过加入 --dry-run 进入演练/预览模式
