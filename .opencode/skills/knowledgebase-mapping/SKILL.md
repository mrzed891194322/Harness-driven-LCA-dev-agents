# 知识库路径映射 (knowledgebase-mapping)

此技能定义了项目中不同目录下的原始文档在转化为 RAG 向量数据库时的目录映射规范，并说明了各输出数据库的来源及用途，以便智能体有针对性地进行数据库构建与查询。

## 知识库构建映射规范

执行 RAG 数据库构建或更新任务时，需要查阅原始目录到目标向量数据库子目录的路径映射规则。
- **参考文件**：具体的目录映射与转换规则已归档在 `assets/mapping-rules.md`。

## 知识库查询检索规范

在调用 `query-rag-database` 技能进行知识库检索前，需要了解各向量数据库的用途以进行针对性查询。
- **参考文件**：各 RAG 向量数据库子目录的具体来源、建议用途已归档在 `assets/knowledge-sources.md`，智能体在查询时应根据其说明针对性地选择数据库目录。
