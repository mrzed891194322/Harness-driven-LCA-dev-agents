# 项目目录规范与文件位置约定 (Project Convention README)

本文件是项目目录结构及文件位置约定的内部路由。智能体在需要查阅文件目录用途或命名规范时，请根据以下场景查阅具体的规范文档：

- **instructions/global-structure.md**：用于查询全局根目录结构（如 `input/`、`output/`、`docs/`）、通用约定、Python 虚拟环境与 `uv` 依赖管理规范。
- **instructions/src-structure.md**：用于查询 `src/` 内部目录结构（如 `eval/`、`history/`、`knowledge/`、`plan/` 等）以及各模块的职责定义。
